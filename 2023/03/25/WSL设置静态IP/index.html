WSL设置静态IP

微软终于更新了 WSL2，允许用户自行指定用于 WSL2 的网卡，因此不再需要任何的脚本等等即可固定 WSL2 地址甚至支持 IPv6 和外部访问。

注意：此方法需要至少 Windows 11 22H2。

首先需要从 Microsoft Store 里面下载最新的 Windows Subsystem for Linux。

装好之后可以运行 wsl --version 确认一下是不是 1.0 以上。
